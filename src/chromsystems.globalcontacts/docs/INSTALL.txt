chromsystems.globalcontacts Installation
----------------------------------------

To install chromsystems.globalcontacts using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``chromsystems.globalcontacts`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        chromsystems.globalcontacts
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
