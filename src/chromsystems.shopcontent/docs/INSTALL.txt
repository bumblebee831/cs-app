chromsystems.shopcontent Installation
-------------------------------------

To install chromsystems.shopcontent using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``chromsystems.shopcontent`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        chromsystems.shopcontent
       
* Re-run buildout, e.g. with:

    $ ./bin/buildout
